# CPU Pathtracer
Simple pathtracing on the CPU written as a learning project back in 2022. I am currently working on a new implementation in C.

> [![YouTube](http://i.ytimg.com/vi/bIIeDLlVScg/hqdefault.jpg)](https://www.youtube.com/watch?v=bIIeDLlVScg)
>
> Demo on YouTube, showing off sphere tracing and the different materials.

## Features
- Sphere tracing.
- Monte carlo sampling with light bouncing.
- Material properties, including metallicity, roughness, emissiveness, and color.
- Simple camera, can't currently look around freely, as I wasn't able to wrap my head around that back in the day.
